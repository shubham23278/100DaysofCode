1. You are given a base b.
2. You are given two numbers n1 and n2 of base b.
3. You are required to subtract n1 from n2 and print the value.

Constraints

2 

Format

Input

A base b
A number n1
A number n2

Output

A number of base b equal in value to n2 - n1.

Example

Sample Input

8
1
100

Sample Output

77
